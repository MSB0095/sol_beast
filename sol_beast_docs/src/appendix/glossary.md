# Glossary

## A

**ATA (Associated Token Account)** - A token account derived from a wallet address and mint address

## B

**Bonding Curve** - Algorithm that determines token price based on supply

**Buy Heuristics** - Rules that determine if a token should be purchased

## D

**Dry Run** - Simulation mode where no real transactions occur

**DEX** - Decentralized Exchange

## H

**Helius Sender** - Ultra-low latency transaction submission service

## M

**Market Cap** - Total value of all tokens (price Ã— supply)

## P

**Priority Fee** - Additional fee to prioritize transaction processing

**Pump.fun** - Solana token launchpad protocol

## R

**RPC** - Remote Procedure Call - API to interact with Solana

**Rug Pull** - Scam where developers abandon project and take funds

## S

**Slippage** - Difference between expected and actual trade price

**Stop Loss** - Automatic sell when position loses too much value

## T

**Take Profit** - Automatic sell when position reaches profit target

**Tip** - Payment to validators/Jito for faster inclusion

## W

**WASM** - WebAssembly - Technology for running code in browsers

**WebSocket** - Real-time bidirectional communication protocol
