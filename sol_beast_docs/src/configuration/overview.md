# Configuration Overview

Sol Beast can be configured through multiple methods:

- **TOML File** (Backend mode) - `config.toml`
- **Environment Variables** - For Docker/cloud deployments
- **UI Settings** (WASM mode) - Browser-based configuration
- **localStorage** (WASM mode) - Persistent browser storage

See the [Settings Reference](./settings-reference.md) for complete documentation of all configuration options.
