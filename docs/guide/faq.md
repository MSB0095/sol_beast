# Frequently Asked Questions

## General Questions

**Q: Is SOL BEAST free to use?**
A: Yes, SOL BEAST is open source under MIT license.

**Q: Do I need a Helius API key?**
A: No, but it's recommended for better performance.

**Q: Can I use this on testnet?**
A: Not currently, but devnet support may be added.

## Trading Questions

**Q: How much SOL do I need to start?**
A: Minimum 0.5 SOL, but 2-5 SOL is recommended.

**Q: What's the difference between dry and real mode?**
A: Dry mode simulates trades without spending SOL. Real mode executes actual transactions.

**Q: Can I run multiple instances?**
A: Yes, but use different wallets for each instance.

## Technical Questions

**Q: What's the minimum system requirements?**
A: 2 CPU cores, 4GB RAM, stable internet. See [Installation](/guide/installation).

**Q: Can I contribute?**
A: Yes! See [Contributing](/guide/contributing).

**Q: Is Windows supported?**
A: Yes, but Linux/macOS is recommended for best performance.
